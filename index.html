    ZubZet – The Modern PHP Framework
    ZubZet is a PHP web framework designed for speed, flexibility, and security. With powerful built-in tools and a streamlined architecture, ZubZet lets you build sophisticated web applications effortlessly.

    Token-Based Security

    Ensures secure and smooth login sessions using token-based authentication.

    Role-Based Permissions

    Assigns precise permissions to users, protecting sensitive actions and data.

    Session Management

    Manages user sessions effectively, ensuring safety across multiple devices.

    Multi-Factor Authentication (MFA) Ready

    Compatible with multi-factor authentication setups for added security.

    Features

  • Auto Form Validation Automate form validation with error reporting and database integration.
  • Permission System Manage roles and permissions with flexible access control, including wildcards.
  • File Uploads Handle file uploads seamlessly with pre-set rules for type and size validation.
  • Email Integration Send emails directly from your application with customizable templates.
  • PDF Generation Generate PDFs from HTML templates, supporting complex layouts and styles.
